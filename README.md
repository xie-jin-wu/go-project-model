# go-project-model

golang项目模板

## 目录结构

- api/
  > http,rpc接口协议
- config/
  > 配置文件操作
- consts/
  > 全局变量和常量
- errors/
  > 自定义错误
- internal/
    - service/
      > http,rpc处理
    - biz/
      > 业务逻辑处理
    - data/
      > 数据保存处理
- pkg/
  > 全局使用方法目录
- sql/
  > sql建表语句
- test/
  > 测试方法
- config.yaml
  > 配置文件
